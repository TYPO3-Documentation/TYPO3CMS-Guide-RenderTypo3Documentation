Introduction

`RenderDocumentation` is a toolchain. It needs to be run by a toolchain runner.
:ref:`TCT <About-TCT>` is that tool.

`RenderDocumentation` does not need a special installation. It is sufficient
to clone the repository. On the server you find it in :file:`~/Toolchains/RenderDocumentation`.
